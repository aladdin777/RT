# RT

* Проект стартует на локальном адресе 127.0.0.1
* Требуется установленный Docker, Ansible и ROOT права.
* Результат в браузере или curl "localhost:80".
* Для запуска проекта выполните следующую команду на локальном сервере
* Тест
#### git clone https://github.com/aladdin777/RT.git && cd RT && sudo ansible-playbook docker.yml
